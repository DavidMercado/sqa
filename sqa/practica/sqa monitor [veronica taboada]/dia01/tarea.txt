2. Que es el User Account Control[UAC]
	Es un componente de seguridad incluido en el S.O. cuyo objetivo es minimizar
	el uso de privilegios de administracion por parte de los usuarios estandar. Tb

3. Introduccion a los archivos de datos de outlook
	Es un lugar de almacenamiento [donde se guardara mensajes de correo
	electronico, el calendario, las tareas y otros elementos] que permite conservar
	los datos en el equipo.

Archivos de datos>
	1. Archivos .ost que se usan cuando se tiene una cuenta en Exchange y se desea
	trabajar sin conexion o cuando se usa modo de intercambio en cache.

	2. Archivos .pst que se usan para cuentas pop3, imap[protocolo de correo de
	internet] y http. Y cuando se desea crear archivos de datos antiguos o copias
	de seguridad de las carpetas y elementos de outlook.

Archivos de carpetas personales[.pst]
	Es un archivo de outlook donde se almacenan mensajes y otros elementos del
	equipo. Necesita conexion a internet.

	Se puede indicar que outlook envie los nuevos elementos al archivo .pst
	pero trae problemas cuando se accede desde otra maquina o desde la web.

Archivos de carpetas sin conexion[.ost]
	Cuando se usa Microsoft exchange server, los mensajes, calendario y otros
	elementos se almacenan en el servidor. Pero se puede indicar que se realice una
	copia local en el equipo. Esto permite trabajar sin conexion y cuando recupera
	la conexion sincroniza con el exchange server.
